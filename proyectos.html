Proyectos Interactivos

Descubre algunos de mis proyectos interactivos y las tecnologías que utilizo para desarrollarlos.

WebApp Interactiva de Gestión de Tareas

Aplicación web para gestionar tareas con funcionalidades como recordatorios, filtros y un modo colaborativo en tiempo real. Usa React, Node.js y MongoDB.

Plataforma de E-learning

Plataforma interactiva para aprender nuevos temas, con videos, cuestionarios y foros de discusión. Desarrollada con React y Firebase para la autenticación y base de datos.

Aplicación de Clima en Tiempo Real

Aplicación móvil que proporciona pronósticos del clima en tiempo real usando la API de OpenWeather. Desarrollada con React Native para una experiencia multiplataforma.

Pagina para crear paginas web

Plataforma para crear paginas web, a tu gustu y con tu diseño.
